Annotator - Responsive UI Demo

Bootstrap 5.x Breakpoints Implementation

Responsive Design: The Annotator application uses Bootstrap 5.x breakpoints to provide an optimal viewing experience across all devices.
Breakpoint Screen Size Device Type Layout Behavior
xs < 576px Portrait phones Single column, collapsed sidebar, hamburger menu
sm ≥ 576px Landscape phones Responsive grid, compact navigation
md ≥ 768px Tablets Fixed sidebar (200px), icons with selective text
lg ≥ 992px Desktops Full sidebar (280px), complete navigation text
xl ≥ 1200px Large desktops Optimized spacing, centered container
xxl ≥ 1400px Extra large desktops Wider sidebar (320px), maximum container width

Responsive Features Implemented

  • Adaptive navigation bar with collapsible menu for mobile devices
  • Responsive sidebar that adjusts width based on screen size
  • Grid system that stacks cards on mobile and displays in columns on desktop
  • Typography scaling for optimal readability across devices
  • Touch-friendly buttons and interactive elements on mobile
  • Contextual text display (hide/show labels based on screen size)
  • Responsive tables with horizontal scrolling on small screens
  • Optimized spacing and padding for different viewport sizes
  • Print-friendly styles that hide navigation elements
  • Accessibility improvements with proper focus states
  • High contrast mode support for better accessibility
  • Reduced motion support for users with vestibular disorders

How to Test Responsive Behavior

Testing Instructions:
  1. Open index.html in your web browser
  2. Use browser developer tools to simulate different device sizes
  3. Resize the browser window to see responsive breakpoints in action
  4. Test navigation functionality on mobile devices
  5. Verify that all content remains accessible and usable at all screen sizes

Files Structure

Quick Start

To run the application locally:
  1. Start a local web server: python3 -m http.server 8080
  2. Open browser to: http://localhost:8080
  3. Test responsive behavior by resizing browser window
